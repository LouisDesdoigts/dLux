    get pixel coordinates

    Returns the physical (x, y) coordinate array of pixel positions relative to the (central) optical axis.

    Parameters:

    Name Type Description Default
    npixels int

    The number of pixels along the side of the output array.

    		 required
    pixel_scale Array, meters

    The physical size of each pixel

    		 required
    x_offset Array

    The x offset of the centre of the coordinate system in the square output array.

    		 np.array(0.0)
    y_offset Array

    The y offset of the centre of the coordinate system in the square output array.

    		 np.array(0.0)

    Returns:

    Name Type Description
    pixel_positions Array

    The (x, y) coordinate arrays in the square output array with the correct offsets.
